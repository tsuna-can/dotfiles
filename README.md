# dotfiles

```shell
# Install git
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install -y git

# Clone dotfiles
$ git clone https://github.com/tsuna-can/dotfiles.git

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ brew bundle
$ make setup-mac
```
